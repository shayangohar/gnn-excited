# gnn-excited

Neural network to analyze excited states in small molecules.

## Structure

- `data/` - local datasets and generated data artifacts. Contents are gitignored.
- `notebooks/` - Jupyter notebooks for exploration and experiments.
- `src/` - dataset classes, model definitions, and training code.
- `configs/` - hyperparameter and experiment configs.

## Getting Started

This repository is initialized as a project scaffold. Add datasets under `data/`, exploratory notebooks under `notebooks/`, and implementation code under `src/`.
